Designing and Creating a Database

  •   Tue 30 October 2018
  •   Data Retrieval and Storing
  •   python SQL SQLite normalization database primary key foreign key

Baseball is a great topic to practice skills in data science. Because of the long history within baseball of the collection and analysis of statistics (most famously Sabermetrics featured in the movie Moneyball), there is a massive potential of answering a pletheora of questions regarding baseball. We will be working with a file of Major League Baseball games from Retrosheet. Retrosheet compiles detailed statistics on baseball games from the 1800s through today. With such a massive amount of data, it will be an ideal project to design and create a relational database which could store and retreive these data efficiently.
