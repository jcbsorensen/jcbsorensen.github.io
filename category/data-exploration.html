Investigating the Differences in Movie Ratings from Fandango

  •   Fri 28 September 2018
  •   Data Exploration
  •   python descriptive statistics exploration population evaluation slicing data frequency distribution table kernel density plot grouped bar plot

In October 2015, a data journalist named Walt Hickey analyzed movie ratings data and found strong evidence to suggest that Fandango's rating system was biased and dishonest (Fandango is an online movie ratings aggregator). He published his analysis in this article - a great piece of data journalism. In this project, we'll analyze more recent movie ratings data to determine whether there has been any change in Fandango's rating system after Hickey's analysis.
